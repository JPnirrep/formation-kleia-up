# Vision produit — Plateforme Formations Kleia-up

## Mission
Héberger, structurer, délivrer et suivre les formations de Kleia-up (Leadership Organique) dans une expérience apprenant moderne, progressive et motivante.

## Positionnement
Ce n'est PAS un simple dépôt de vidéos. C'est une véritable plateforme d'apprentissage avec :
- Parcours pédagogiques structurés (modules → leçons → vidéos → quiz)
- Suivi de progression fiable avec reprise automatique
- Certificats de complétion
- Dashboard apprenant personnalisé
- Préparation à l'IA (recommandations, analytics, recherche transcript)

## Cibles
- **B2C** : Entrepreneurs HPI/HSP, coachés individuels de Sandrina Perrin
- **B2B** (futur) : Entreprises clientes, cohortes — reporté après V1

## Différenciation
- Design cohérent avec le site kleia-up.fr (même identité visuelle)
- Progression fiable (pas de "vidéo ouverte = terminée", seuil configurable 85%)
- Accessibilité WCAG 2.2 dès la V1
- Architecture modulaire extensible vers IA, analytics, interopérabilité
